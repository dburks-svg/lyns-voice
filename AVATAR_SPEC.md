# ARCHITECTURAL SPECIFICATION: REAL-TIME 3D DESKTOP AVATAR OVERLAY

**Target Environment:** Claude Code CLI (Max Subscription)

**Voice Bridge:** `mcp-voice-hooks` (Browser-Native Web Speech API)

**Visual Layer:** Three.js Custom Shader Mesh (Gary Simon / Jarvis Vibe)

---

## 1. Project Objective & Scope

The objective is to overlay a stylized, real-time 3D pulsing blue mesh head onto the browser-native web application generated by the `mcp-voice-hooks` Model Context Protocol (MCP) server.

```
[Claude Code CLI] ──(Max Subscription)──> [mcp-voice-hooks Server (Port 5111)]
                                                          │
  [Three.js Pulsing Avatar] <──(Speech Synthesis Hooks)───┴──> [Browser Frontend UI]

```

The system must run 100% locally for audio processing using the browser's native **Web Speech API** (Speech-to-Text and Text-to-Speech), ensuring zero additional API costs, while retaining full reasoning capabilities via the Claude Max subscription.

---

## 2. Core Technical Architecture

| Layer | Component | Implementation Specification |
| --- | --- | --- |
| **Orchestration** | Claude Code CLI | Processes incoming voice tokens textually through the Max plan framework. |
| **Middleware** | `mcp-voice-hooks` | Intercepts standard lifecycle events and hosts a local HTTP server on `http://localhost:5111`. |
| **Frontend Renderer** | Three.js (WebGL Canvas) | Renders a 3D wireframe mesh head or abstract digital orb on a dark transparent-friendly layout. |
| **Audio-Visual Hook** | `SpeechSynthesis` Hooks | Leverages `onboundary` word markers and `AnalyserNode` frequency spectrums to mutate 3D geometry in real time. |

### Phase 2: Construct the Blue Pulsing Mesh Head
Create an isolated JavaScript module (`avatar.js`) to handle the WebGL canvas initialization:
*   **Geometry:** Create a stylized low-poly `IcosahedronGeometry` or a detailed 3D head mesh.
*   **Material:** Use a `MeshBasicMaterial` or a custom `ShaderMaterial` with `wireframe: true`. Color scheme must match an emissive neon cyan/blue (`#00f0ff` or `#0077ff`).
*   **Idle Animation:** Implement a continuous, gentle noise-based vertex deformation (e.g., using simplex noise) to give the head a living, breathing appearance while waiting for input.

### Phase 3: Bind Voice Output to Mesh Deformation
Intercept the system's text-to-speech engine to make the head respond to Claude's actual speech cadence:
1. Locate the native browser `window.speechSynthesis.speak(utterance)` handler in the `mcp-voice-hooks` client source.
2. Bind to the `onboundary` event hook of the active `SpeechSynthesisUtterance`.
3. When `event.name === 'word'` is fired, trigger an instantaneous displacement impulse on the mesh vertices or dramatically spike the scale of the object.
4. Alternatively, use a browser `AudioContext` loop to capture the audio output stream and pass real-time amplitude data (`getByteFrequencyData`) directly into a custom vertex shader uniform to warp the mesh dynamically with audio frequencies.

### Phase 4: Styling and UI Polish
*   Apply dark mode CSS elements to make the background entirely black or transparent-ready.
*   Center the Three.js canvas so the glowing digital head occupies the focal point of the browser window.
*   Ensure layout responsiveness so the window can be shrunk into a small floating side-panel on the desktop layout next to the terminal CLI.

## 4. Expected Behavior Rules

*   **State: Idle** $\rightarrow$ Head pulses with low frequency and slow ambient rotation. Glowing intensity remains at baseline (dark navy/slate blue color spectrum).
*   **State: Listening (User Speaking)** $\rightarrow$ Mesh rhythmically distorts or compresses vertically to show active audio feedback from the microphone data stream.
*   **State: Thinking (Processing)** $\rightarrow$ Mesh pulses rapidly or shows a circular "orbital pattern" to indicate Claude Code is awaiting internal tool execution/inference.
*   **State: Speaking (Claude Replying)** $\rightarrow$ The vertex displacements match the spoken phonemes and word boundaries in real time, projecting an intense bright blue neon glow.
